## Demo:

[sublime.npmstack.com](//sublime.npmstack.com)

### Setup

```
git clone https://github.com/sethbergman/sublime.git && cd sublime
```

### Install dependencies

```sh
npm i # or use yarn
yarn
```

### Start the app

```
npm start
```

##### Visit [http://0.0.0.0:5000](http://0.0.0.0:5000) in your browser.
